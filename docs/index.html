AI 기반 전국 교량 노후도 및 안전점검 우선순위 분석

전국 교량 공개 데이터를 활용한 머신러닝·딥러닝 포트폴리오 프로젝트

안내

본 분석은 공개 데이터 기반의 점검 우선순위 후보 분석이며, 실제 교량 안전진단을 대체하지 않는다.

프로젝트 개요

국토교통부 전국교량표준데이터를 기반으로 교량의 노후도, 점검 이력, 구조 특성, 내진 관련 정보를 정리하고 점검 우선순위 후보를 분석했다. 규칙 기반 라벨을 분류 모델로 비교하고 AutoEncoder로 일반적인 패턴에서 벗어나는 후보를 탐색했다.

한계점

priority_score와 target_priority는 분석 목적의 보조 지표다. 실제 구조 안전성 판단에는 전문 진단 데이터, 현장 점검, 구조해석 정보가 필요하다. 모델 성능은 실제 위험 예측력이 아니라 규칙 기반 라벨 재현 성능으로 해석해야 한다.
